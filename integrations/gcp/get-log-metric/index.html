gcp get-log-metric

Get log metric

Description

Get log metric for gcp logs

Synopsis

gcp get-log-metric
    [--site <site>]
    [<name>]

Arguments

site - (string)

     Site where this command will be executed
     Example: --site "site-1"
     Default: input.site      Attributes: optional

name - (string)

     Metric name to fetch metric
     Example: "CassandraMetric"
     Default: _None_      Attributes: optional
