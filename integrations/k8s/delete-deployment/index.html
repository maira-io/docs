k8s delete-deployment

delete deployment

Description

delete deployment

Synopsis

k8s delete-deployment
    [--site <site>]
    [--cluster <cluster>]
    [--namespace|-n <namespace>]
    <name>

Arguments

site - (string)

     Site where this command will be executed
     Example: --site "site-1"
     Default: input.site      Attributes: optional

cluster - (string)

     Cluster where this command will be executed
     Example: --cluster "in-cluster"
     Default: in-cluster      Attributes: optional

namespace - (string)

     Kubernetes namespace
     Example: --namespace "namespace-1"
     Default: default      Attributes: optional

name - (string)

     Name of the deployment to be deleted
     Example: "name-1"
     Default: _None_      Attributes: required

Examples

Input: 

! k8s delete-deployment "kafka2"
Output: 
deployment kafka2 deleted
