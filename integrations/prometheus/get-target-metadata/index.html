prometheus get-target-metadata

Get metadata about metrics scraped from targets

Description

Returns metadata about metrics currently scraped from targets

Synopsis

prometheus get-target-metadata
    [--site <site>]
    [--cluster <cluster>]
    [--label <label>]
    [--label_value <label_value>]
    [--metric <metric>]
    [--limit <limit>]

Arguments

site - (string)

     Site where this command will be executed
     Example: --site "site-1"
     Default: input.site      Attributes: optional

cluster - (string)

     Name of prometheus cluster
     Example: --cluster "prometheus-default"
     Default: prometheus-default      Attributes: optional

label - (string)

     Name of Label selector used to match targets
     Example: --label "job, instance"
     Default: _None_      Attributes: optional

label_value - (string)

     Value of Label selector used to match targets
     Example: --label_value "prometheus, 127.0.0.1:9000"
     Default: _None_      Attributes: optional

metric - (string)

     A metric name to retrieve metadata for
     Example: --metric "go_goroutines"
     Default: _None_      Attributes: optional

limit - (int)

     Maximum number of targets to match
     Example: --limit 2
     Default: _None_      Attributes: optional

! prometheus get-target-metadata --site "localhost" --label "job" --label_value "prometheus"
