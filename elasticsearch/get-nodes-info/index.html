    elasticsearch get-nodes-info

    Get elastic search cluster nodes information

    Description

    Retrieves one or more (or all) of the cluster nodes information.

    Synopsis

    elasticsearch get-nodes-info [--site <site>] [--cluster <cluster>] [--nodes <nodes>]

    Arguments

    site - (string)

         Site where this command will be executed

         Example: --site site-1
         Default: input.site
         Attributes: optional

    cluster - (string)

         Name of elastic search cluster

         Example: --cluster elastic-1
         Default: default
         Attributes: optional

    nodes - (string)

        

         Example: --nodes node1
         Default: None
         Attributes: optional, multiple allowed

    Examples

    Input: 

    ! elasticsearch get-nodes-info --site "localhost" --cluster "elastic-1" --nodes "beehyv-H81M-S"
