    elasticsearch nodes-info

    Get cluster nodes information

    Description

    Retrieves one or more (or all) of the cluster nodes information.

    Synopsis

    elasticsearch nodes-info
    [--site <site>]
    [--cluster <cluster>]
    [--filter_path <filter_path>]
    [--raw]
    [<nodes>]
    [--metric <metric>]

    Arguments

    site - (string)

    Site where this command will be executed
    Example: --site "site-1"
    Default: input.site Attributes: optional

    cluster - (string)

    Name of elastic search cluster
    Example: --cluster "elastic-default"
    Default: elastic-default Attributes: optional

    filter_path - (string)

    Path in the response. Filters and returns only parts of response matching this path
    Example: --filter_path "index.mappings.properties.*"
    Default: _None_ Attributes: optional, multiple allowed

    raw - (bool)

    If false, returns compact and aligned text output
    Example: --raw
    Default: _None_ Attributes: optional

    nodes - (string)

    Nodes
    Example: "node1"
    Default: _all Attributes: optional, multiple allowed

    metric - (string)

    Limits the information returned to the specific metrics
    Example: --metric "http, ingest, jvm"
    Default: _all Attributes: optional, multiple allowed
