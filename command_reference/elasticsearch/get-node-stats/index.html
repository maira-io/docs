    elasticsearch get-node-stats

    Get elastic search node statistics

    Description

    Returns node statistics.

    Synopsis

    elasticsearch get-node-stats
    [--site <site>]
    [--cluster <cluster>]
    [--filter_path <filter_path>]
    [--nodes <nodes>]
    [--metrics <metrics>]
    [--index_metric <index_metric>]

    Arguments

    site - (string)

         Site where this command will be executed
         Example: --site "site-1"
         Default: input.site      Attributes: optional

    cluster - (string)

         Name of elastic search cluster
         Example: --cluster "elastic-default"
         Default: elastic-default      Attributes: optional

    filter_path - (string)

         Path in the response. Filters and returns only parts of response matching this path
         Example: --filter_path "index.mappings.properties.*"
         Default: _None_      Attributes: optional, multiple allowed

    nodes - (string)

         List of node IDs or names used to limit returned information.
         Example: --nodes "node-1"
         Default: _None_      Attributes: optional, multiple allowed

    metrics - (string)

         Limits the information returned to the specific metrics
         Example: --metrics "breaker, discovery, http, indices, ingest"
         Default: _None_      Attributes: optional, multiple allowed

    index_metric - (string)

         Limit the information returned for indices metric to the specific index metrics. It can be used only if indices (or all) metric is specified
         Example: --index_metric "completion, docs, fielddata, flush, indexing, merge, query_cache"
         Default: _None_      Attributes: optional, multiple allowed
