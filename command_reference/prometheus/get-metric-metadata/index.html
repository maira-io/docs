    prometheus get-metric-metadata

    Get metadata about metrics currently scrapped from targets

    Description

    Returns metadata about metrics currently scrapped from targets. However, it does not provide any target information

    Synopsis

    prometheus get-metric-metadata
    [--site <site>]
    [--cluster <cluster>]
    [--metric <metric>]
    [--limit <limit>]

    Arguments

    site - (string)

         Site where this command will be executed
         Example: --site "site-1"
         Default: input.site      Attributes: optional

    cluster - (string)

         Name of prometheus cluster
         Example: --cluster "prometheus-default"
         Default: prometheus-default      Attributes: optional

    metric - (string)

         A metric name to filter metadata for
         Example: --metric "http_requests_total"
         Default: _None_      Attributes: optional

    limit - (int)

         Maximum number of metrics to return.
         Example: --limit 2
         Default: _None_      Attributes: optional
